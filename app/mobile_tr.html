모바일 진위 확인

온라인 발급 민원서류를 별도의 PC 또는 온라인 접속 없이 스마트폰 앱에서 2차원 바코드 인식으로 검증가능합니다.

모바일 검증 서비스

주요 특장점

  • 1. 기존 인터넷증명서 발급 시스템 그대로 사용
  • 2. 고객 PC에 2차원 바코드 발급 모듈(Client 모듈)만 재설치 (3단 분할코드 생성 모듈 설치)
  • 3. 현재 서비스 중인 모든 증명서 적용 가능
  • 4. 스마트폰 이외의 부가장비(스캐너 등) 불필요
  • 5. 실시간 확인 (인터넷이 되지 않는 곳에서도 확인 가능)
  • 6. 기존 평판스캐너를 통한 PC 위변조 확인 가능 (평판스캐너용 프로그램 재배포)
