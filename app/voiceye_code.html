VOICECode

VOICECode는 WEB 또는 WAS 서버에 바코드 생성 엔진이 구축되어 보이스아이 바코드 이미지를 생성해 주는 솔루션 입니다. 리포팅툴 및 HTML 뷰어 연계를 통한 각종 서식에 적용이 가능합니다

특징

국가표준 관련 법령
특장점 세부사양
정보접근성 향상
  • 국내 유일의 시각장애인용 음성 변환 2차원 바코드 표준 규격 지원
  • 방송통신 국가표준 '시각장애인의 인쇄물 접근성 향상을 위한 점자·음성 변환용 코드 활용지침'(KS X3095)준수
  • 정보통신 단체표준 '시각장애인용 AD 2차원 바코드'(TTAK,KO-06.0180), '시각장애인 인쇄물 음성 변환 출력용 2차원 바코드 규격'(TTAK.KO-06.081) 준수
안정성
  • GS인증 제품으로 국내·외 몇 종의 다수 특허 보유
  • 바코드 이미지 용량이 매우 작기 때문에 서버에 대한 부하 없음
  • 전국민을 대상으로 하는 G4C 전자정부 민원서류, 대법원 판결문 등에서 이미 검증 받은 솔류션
APP
  • 스마트폰 APP을 이용하여 음성으로 호환 기능
  • 고령자를 위한 독서 확대 기능
  • 다문화 가정 및 외국인 관광객을 위한 번역 기능(구글 번역 엔진 사용)
호환성
  • 다양한 리포팅 툴과의 연동 지원
  • 음성 변환용 2차원 바코드는 현재 출시된 인쇄물 음성변환출력기와 100% 호환되어 음성으로 출력
