VOICEYE Maker(S/W)

인쇄물 우측 상단에 함께 인쇄될 보이스아이 코드 이미지를 자동 생성해주는 PC용 소프트웨어 프로그램입니다.

  • For HWP
  • For MS-Word
  • For InDesign
  • For Quark
  • For Server/Client

VOICEYE Maker for HWP

VOICEYE Maker for HWP는 HWP 문서의 페이지 우측 상단에 보이스아이 코드를 자동으로 생성/삽입해주는 프로그램입니다.
스마트폰 또는 전용 리더기로 보이스아이 코드를 스캔하면, HWP 문서 내용을 확대해서 보거나 소리로 들을 수 있습니다.

VOICEYE Maker(S/W)는 상담을 통한 접수 후 구매 가능합니다.
TEL: 02-811-3760 / E-Mail: sales@voiceye.com

메뉴얼 다운로드

VOICEYE Maker for HWP는 HWP 문서의 텍스트를 자동 추출하여 보이스아이 코드를 쉽게 생성하고, 원본과 함께 보이스아이 코드가 삽입된 문서를 출력할 수 있도록 해주는 소프트웨어 프로그램입니다.
보이스아이 메이커는 HWP에 플러그인 형태로 설치되며, HWP 실행 시, 메뉴바에 "보이스아이"라는 메뉴 아이콘이 추가됩니다

주요 기능

  • 원본 텍스트 추출
  • 명함용 보이스아이 코드 생성
  • 보이스아이 코드에 저장될 내용 미리 듣기
  • 음성으로 듣기에 자연스러운 내용으로 편집
  • 지정된 위치에 보이스아이 코드 삽입/출력
  • 한 번의 클릭으로 모든 페이지에 보이스아이 코드 자동 생성/삽입
  • 보이스아이 라벨 및 노트 생성
  • 텍스트 데이터 압축 및 보이스아이 코드 생성/저장

지원 사양

  • Pentium IV 1.8GHz 프로세서
  • 512MB RAM (Windows Vista의 경우 최소 1GB)
  • Windows XP, Vista or Windows7
  • HWP 2002-2010

Server Engine

주요 기능

  • WEB 또는 WAS 서버에 바코드 생성 엔진이 구축되어 바코드 이미지를 생성 해주는 Engine이다.
  • Report Viewer가 활용되지 않는 민원서류 또는 증명서에 보이스아이를 적용하기 위해 서버에서 바코드이미지를 생성 할 수 있도록 하는 솔루션

주요 커스트마이징 범위

  • 인터페이스(JNI) 수정
    • DB로부터 Query된 Data를 음성변환용 2차원 바코드 엔진으로 전달하기 위한 파라미터 Input용 API 수정
    • 필요 시 별도의 안내 멘트 또는 텍스트 내용 변경 시 관련 필드값 변환 처리를 위한 API 수정
  • Report viewer 보고서 수정
    • 보이스아이 코드 이미지가 민원 서식의 우측 상단 (표준 위치)에 삽입되기 위해 Report Viewer의 보고서를 수정해야 함.

Client Engine

주요 기능

  • 보이스아이 메이커 Client Engine은 Client PC에 설치되어 바코드 이미지가 생성되며, 주로 레포팅 툴과 연동되거나 C/S용 프로그램에 구동되어 각종 민원서식에 활용됨.

주요 커스트마이징 범위

  • Report viewer 보고서 수정
    • DB로부터 Query된 Data를 음성변환용 2차원 바코드 엔진으로 전달하기 위해 Viewer에서 텍스트 Data를 엔진으로 전달하기 위한 파라미터 Input용 API 수정
    • 생성된 코드 이미지를 레포트에 삽입하기 위한 API 추가 및 보고서 레이아웃 수정
  • 한글 포맷의 경우에는 한글 OCX, 레포터와 PDF 등의 뷰어 형태의 서식인 경우에는 라이브러리 형태로 제공되어 2차원 바코드를 생성.
